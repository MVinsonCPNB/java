# FlashCard_App Usage

# First open up the QuizCardBuilder

$java QuizCardBuilder

    --Enter a Question 
    --Enter the Answer
    --Press the Next Card button to enter more cards

    --Select File > Save > save the Questions and Answers to a file


# Open up QuizCardPlayer

$java QuizCardPlayer

    --Select File > Load card set > select the saved Questions and Answers file
    --Press the Show Answer/Show Question button till the end
